	用 "|" 管道线可以将多个简单的命令集合在一起，用以完成较复杂的功能

Linux基本操作指令：
	对目录进行的操作命令
		命令的使用格式：
			命令名 [选项] [源操作对象] [目标操作对象]
			ls 列出目录下的内容
			ls -a 查看所有文件，包含隐藏文件。以'.'开头的文件是隐藏文件
			-l 查看详细信息
			-t 按照时间顺序排序
			-r按照逆序排序
			-S按文件大小排序
			file：文件命名指令
			mkdir: 创建文件夹命令，-p 递归创建
			rmdir：删除空目录
			man [命令]：命令手册
			pwd 显示当前路径
			. 代表当前目录
			.. 代表上一层目录
			wc：用来统计给定文件的行数/字数/字符数，其格式为：
			wc [-lw] [-c] 文件名
			查找出来为行数，字数，字符数和文件名
			history:查看所有历史命令
			ln[选项] 源文件 [目标文件]：文件的链接命令
		Linux下的目录是一个树形结构
		绝对路径：/home/lyc/aaa.txt  起始查找路径是根目录/以根目录为起始查找路径
		相对路径：../lyc/aaa.txt  /起始查找路径是当前路径/从当前路径开始查找

Linux下一切皆文件，Linux下并不以文件的后缀作为文件名

lyc@skrskr~    当前用户名@当前主机名   ~:当前用户的家目录

rm 	默认删除文件
	-f 强制删除文件
	-r 递归删除，要求确认
cp	拷贝文件到指定位置
	默认是拷贝普通文件
	-r  拷贝所有文件包含目录型文件
	-f  忽略提示信息
mv	移动指定文件到指定位置 ，有着一个剪切的效果
	-i 增加提示信息
文件的操作命令：
touch	若文件不存在则创建空文件，存在则刷新文件时间属性（很少用）
	-r 以一个文件的时间属性来刷新另一个文件
	-t 以指定的时间来刷新文件时间属性[[CC]YY]MMDDhhmm[.ss]
cat	打印文件的内容在终端显示
	-n 显示行号
tac	按行逆序打印文件内容在终端显示
more	按页显示文件内容，B向上，空格向下，回车按行走，q退出
less	按页显示文件内容，f，空格向下翻页，b向上翻页，上下按行走，q是退出 
	/string向下匹配  ?string 向上匹配字符串
head	显示文件的前n行内容，默认为十行。
	-n 3	指定显示前3行
tail	显示文件的后n行内容，默认为十行。
	-n 3	指定显示后3行
压缩操作的命令：
	压缩/解压缩/打包
	zip/unzip	按zip格式进行压缩/解压缩文件 .zip
	gzip/gunzip 按gzip格式进行压缩/解压缩文件 .gz
	bzip2/bunzip2	按bzip2格式进行压缩/解压缩文件 .bz2
	tar 文件打包	可对目录打包，也可对多个目录打包
	-c 打包  -x解包  -v显示详细的打包/解包信息  -f 指定包名，所以-f通常是最后一个选项
	-z 打包/解包同时以gzip格式压缩/解压缩 .tar.gz
	-j  打包/解包同时以bzip2格式压缩/解压缩	.tar.bz2
查找匹配操作命令：
	find文件查找
	find  路径	-name “文件名”文件名可以使用通配符*
		-type	按文件类型查找
		-size  +/-n[cwbkMG]     按文件大小查找
		-mmin	按最后修改时间按分钟查找
		-mtime	按最后修改时间按天来查找
	grep字符串内容匹配
	        在文件内容中匹配包含指定字符串的行
	        -i 忽略大小写进行匹配
	        -v 反向匹配，不包含字符串的行
	        -R “文件名”路径    递归在指定目录下的所有文件中进行匹配
